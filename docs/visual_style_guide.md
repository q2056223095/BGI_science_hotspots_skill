# Visual Style Guide｜小红书轮播图视觉规范

## 1. 总体风格

本仓库图片风格定位为：

**专业可信 + 小红书可读 + 轻科技感 + 信息图表达。**

不是普通海报，也不是纯科研论文图，而是把科研事实转成大众能理解的视觉说明。

## 2. 尺寸

推荐：

- 3:4 竖版
- 适合小红书轮播
- 每张图独立成图，不拼成长图

## 3. 色彩

核心色彩：

- 深蓝：用于标题、科技背景、重要框架
- 科技蓝：用于连线、数据、发光结构
- 白色 / 浅灰：用于健康类、解释类页面背景
- 亮黄 / 橙色：仅用于重点词、编号、强调结论

避免：

- 大面积高饱和红色
- 电商橙红渐变
- 杂乱多色

## 4. 字体与排版

- 标题必须大，适合手机端一眼读懂
- 每页只表达一个核心结论
- 不建议每页超过 3 层信息
- 大标题 + 图示 + 结论框 是推荐结构

## 5. 三类图片风格

### A. 品牌主线科普：鲸落类

特征：

- 深海背景
- 实拍感 / 科研纪实感
- 生物关系、生态网络、生命周期
- 图注要有科学来源意识

适合主题：

- 鲸落
- 生物多样性
- 物种生命周期
- 自然生命课堂

### B. 健康转化科普：肌肉类

特征：

- 白底 / 浅蓝底
- 人体结构示意
- 系统关系：代谢、免疫、炎症、细胞通讯
- 图标化、清晰分块

适合主题：

- 精准健康
- 运动处方
- 营养
- 衰老机制
- 健康管理能力

### C. 教育观点：JUNO / AI时代人才类

特征：

- 深蓝科技感
- 大科学装置 / 数据屏 / 复杂系统
- 能力拆解图
- 青少年真实问题探索场景

适合主题：

- AI时代人才选拔
- 科研实践
- 专业选择
- 跨学科能力
- 大科学工程

## 6. 图片页面结构建议

### P1 封面
核心问题 + 强视觉主体 + 角标

### P2 反常识
传统理解 vs 科学理解

### P3 机制 / 装置
核心科学事实可视化

### P4 结构拆解
流程、系统、能力、关系

### P5 收束
教育价值 / 健康管理价值 / 未来能力

## 7. 禁止项

- 不要把 5 张图拼成一张
- 不要用低质卡通堆图
- 不要出现疗效承诺、药瓶、处方、夸张病痛场景
- 不要用真实小朋友照片
- 不要让文字过密，导致手机端不可读
