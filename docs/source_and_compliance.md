# Source and Compliance｜事实来源与合规规则

## 1. 来源优先级

优先使用：

1. 原始论文
2. 科研机构官网
3. 权威科普机构
4. 主流媒体科学报道

谨慎使用：

- 自媒体二次解读
- 没有来源的截图
- 只引用标题不看正文的内容

## 2. 科学表述

推荐措辞：

- 研究提示
- 研究发现
- 为理解……提供了机制线索
- 可能参与
- 在模型中观察到

避免措辞：

- 已经证明
- 一定会
- 可以治疗
- 能防癌
- 彻底逆转

## 3. 健康类内容

必须避免：

- 医疗诊断
- 个体化用药 / 补剂建议
- 治疗承诺
- 疾病恐吓
- 将动物模型直接外推到人

建议加注：

```text
科普内容仅用于健康认知，不作为个体诊疗建议。
```

## 4. 教育类内容

避免：

- 夸大升学效果
- 承诺竞赛 / 录取 / 证书结果
- 空泛地说“提升能力”

建议拆解能力：

- 会提问
- 会用工具
- 会判断数据
- 会验证结论
- 会协作解决真实问题

## 5. 图片来源与生成

如果直接使用论文 / 机构图片：

- 需要标注图源
- 注意版权与商用限制

更稳妥的做法：

- 基于论文信息科学重绘
- 保留科学结构，不直接搬运原图

## 6. 小红书发布建议

建议在笔记末尾保留：

```text
内容基于公开科研报道整理，科普内容不替代专业诊断或个体化建议。
```

健康类建议尤其保留这句话。
