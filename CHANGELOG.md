# Changelog

## 0.4.0

### 核心定位调整

本版本将 skill 从“教育 / 品牌转化型科学热点内容工具”调整为“科学热点小红书科普转译工具”。

核心原则调整为：

```text
先做科普，不做教育转化。
```

除非用户明确要求，否则不再把科学热点强行转向：

- 孩子成长
- 家长教育
- 科学素养培养
- 未来能力
- 课程价值
- 机构品牌表达
- 活动、直播、报名、课程入口

### 内容方向更新

原三类内容方向：

```text
品牌主线 / 健康转化 / 教育观点
```

更新为四类科普方向：

```text
发现型 / 机制型 / 技术型 / 争议反常识型
```

### 文案结构更新

小红书正文结构更新为：

```text
开头抓人
→ 简短交代新闻事实
→ 用大白话解释科学点
→ 讲清为什么重要
→ 保留科学边界
→ 克制收束
```

删除默认的教育启发、课程转化和品牌连接结构。

### 视觉规则更新

视觉参考从旧的：

```text
鲸落组 / 肌肉组 / JUNO组
```

更新为：

```text
发现型 / 机制型 / 技术型 / 争议反常识型
```

并要求每次生成图片前先完成视觉参考选择。

### 合规规则更新

强化以下边界：

- 不把机制研究写成生活建议
- 不把初步研究写成确定结论
- 不把早期技术写成已落地应用
- 不把“修正认知”写成“彻底推翻”
- 不强行加入教育、课程或品牌转化
