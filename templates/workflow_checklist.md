# 工作流检查清单

## A. 选题判断
- [ ] 这条热点属于品牌主线 / 健康转化 / 教育观点哪一类？
- [ ] 它最适合服务华大教育中心的哪个目标？
- [ ] 它和用户真正关心的问题是否已连接？

## B. 文案检查
- [ ] 是否不是单纯新闻复述？
- [ ] 是否有清晰的反常识开头？
- [ ] 是否把复杂科学讲成了大白话？
- [ ] 是否自然连接了华大教育中心的价值？
- [ ] 是否有轻行动号召？

## C. 图片检查
- [ ] 是否为 5 张独立图，而不是拼成长图？
- [ ] 是否适合小红书 3:4？
- [ ] 每张图是否只有一个核心结论？
- [ ] 是否保持统一视觉风格？
- [ ] 是否有明显的信息层级？

## D. 合规检查
- [ ] 健康类内容是否避免医疗承诺？
- [ ] 研究结论是否被夸大？
- [ ] 教育观点是否落到了具体能力？
- [ ] 是否避免了过强销售感？

## E. 输出交付
- [ ] 标题是否足够可选（至少 5 个）？
- [ ] 正文是否可直接发？
- [ ] 图片规划是否明确？
- [ ] 如需出图，是否为 5 张独立成图？
