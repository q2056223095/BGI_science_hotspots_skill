# 图片生成模板（小红书 5 张独立图）

## 通用规格

- 比例：3:4 竖版
- 用途：小红书轮播图
- 风格：专业、可信、现代、轻科技感、信息图风格
- 文字：大标题醒目，结论前置，分块明确
- 颜色：深蓝 / 科技蓝 / 白底 / 少量亮黄
- 每张图只表达一个核心观点
- 手机端可读，不要文字过密

## 不同内容类型的视觉方向

### 发现型科普文

视觉关键词：

- 真实发现感
- 自然 / 深海 / 化石 / 地球历史 / 生态网络
- 科研纪实感
- 时间线或证据链

提示词参考约束：

```text
视觉参考优先对齐 assets/ 中的古生物、深海生态、生物多样性、自然发现类参考图，保持科研纪实感、清晰信息图结构和真实发现感。
```

---

### 机制型科普文

视觉关键词：

- 人体机制
- 脑科学 / 健康机制 / 生理过程
- 信息卡片
- 时间轴 / 流程图 / 结构图

提示词参考约束：

```text
视觉参考优先对齐 assets/ 中的生命机制、健康管理、脑科学、人体机制类参考图，保持专业可信、结构化信息图风格，避免医疗广告感和夸张疗效感。
```

---

### 技术型科普文

视觉关键词：

- 深蓝科技感
- 数据分析
- 大科学装置
- 材料筛选
- 仪器系统
- AI + 科研

提示词参考约束：

```text
视觉参考优先对齐 assets/ 中的 AI 科研、大科学装置、数据分析、材料科学、科技系统类参考图，保持深蓝科技感、信息图结构和专业科研气质。
```

---

### 争议 / 反常识科普文

视觉关键词：

- 旧认知 vs 新发现
- 证据链
- 时间线修正
- 认知转折
- 对比结构

提示词参考约束：

```text
视觉参考优先对齐 assets/ 中的认知转折、科学证据链、旧认知 vs 新发现、机制拆解类参考图，保持清晰对比结构和克制的科学表达。
```

## 图片质量自查

每组图片生成后，必须检查：

1. 是否明显参考了 `assets/` 中同类型参考图？
2. 是否保持统一视觉气质？
3. 是否每张图都像同一套轮播图？
4. 是否存在过度 AI 感、过度炫光或廉价科技感？
5. 是否信息层级清楚，手机端可读？
6. 是否与正文内容一致？
7. 是否避免了电商风、低幼风、医疗广告风？
8. 是否避免了夸张疗效、恐吓和标题党视觉？
