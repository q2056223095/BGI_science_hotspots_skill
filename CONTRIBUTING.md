# Contributing Guide

这个仓库用于沉淀“华大教育中心科学热点转译”内容方法，不是普通素材堆放库。新增内容时，请保持以下标准。

## 1. 新增热点案例

在 `examples/` 下新增 Markdown 文件，命名建议：

```text
04_topic_name.md
```

每个案例至少包含：

- 内容类型：品牌主线 / 健康转化 / 教育观点
- 题目方向
- 核心来源
- 文案核心观点
- 小红书正文
- 5 张轮播图规划
- 合规提醒

## 2. 新增图片资产

图片放入 `assets/`，命名建议：

```text
主题_页码_页面核心结论.png
```

例如：

```text
鲸落_01_深海王国封面.png
肌肉流失_03_细胞外囊泡机制.png
JUNO_05_AI时代能力收束.png
```

新增图片后，请同步更新：

```text
assets/ASSET_INDEX.md
```

## 3. 文案标准

请优先遵循 `SKILL.md` 中的结构：

```text
热点事实 → 用户真正关心的问题 → 科学机制 / 方法 → 华大教育中心教育价值 → 轻转化入口
```

避免：

- 新闻搬运
- 过度煽情
- 硬广
- 医疗承诺
- 空泛教育鸡汤

## 4. 图片标准

图片必须满足：

- 小红书竖版 3:4
- 每页一个核心结论
- 深蓝 / 科技蓝 / 白底 / 少量亮黄
- 信息图结构清楚
- 与 `assets/` 中既有风格一致

## 5. 事实与合规

健康类内容尤其要注意：

- 不替代诊断
- 不给个体治疗建议
- 不写“防癌”“治病”“逆转疾病”等绝对化表达
- 明确研究性质：机制研究 / 动物实验 / 人群研究 / 科普报道

教育类内容要注意：

- 不空喊能力
- 必须拆解成具体能力：提问、工具、数据、判断、协作、解决真实问题
