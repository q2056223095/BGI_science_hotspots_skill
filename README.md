# BGI Science Hotspots Skill

> 科学热点小红书科普转译技能包：把科学新闻、论文成果和科研发现，转化为准确、克制、好读、适合小红书传播的科普内容。

本仓库用于沉淀一套可持续复用的内容生产方法，帮助内容团队将科学热点转化为：

- 小红书科普文案
- 科学热点背后的关键事实与机制解释
- 5 张独立小红书轮播图规划
- 统一视觉风格参考图
- 去 AI 味人工编辑后的真实发布稿
- AI 味自查自纠评分与发布前质检
- 可复用的选题、文案、图片和合规工作流

---

## 一句话定位

这不是“追热点”的工具，而是一个把科学热点转化为：

```text
准确事实 + 清楚解释 + 真实表达 + 科学边界 + 小红书传播感
```

的内容生产 Skill。

它的核心目标不是快速生成一篇“看起来完整”的科普文，而是形成一套稳定工作流：

```text
科学热点识别
→ 事实核验
→ 内容转译
→ 去 AI 味人工编辑
→ AI 味自查评分
→ 不达标重写
→ 小红书最终发布稿
→ 轮播图视觉规划
→ 图片生成提示词
→ 合规检查
```

重要原则：

```text
先做科普，不做教育转化。
```

除非用户明确要求，不要把每个选题都强行引申到孩子成长、家长教育、课程价值、活动报名或机构品牌表达。

好的科普本身就能建立信任。

## 快速开始

### 方法 1：直接使用主技能文件

复制：

```text
SKILL.md
```

作为模型的长期规则。

适合场景：

- 想让模型稳定按照“准确科普 + 小红书表达”的方式输出
- 要同时产出文案 + 图片规划
- 要保持同一套栏目感和视觉风格
- 要降低文案的 AI 味、模板感和标题党感
- 要避免把科学热点强行转成教育、课程或品牌广告

---

### 方法 2：使用总提示词

复制：

```text
prompts/master_prompt.md
```

然后把新的科学热点贴进去即可。

---

### 方法 3：按已有案例改写

参考：

- `examples/01_whale_fall.md`：发现型科普文
- `examples/02_muscle_loss.md`：机制型科普文
- `examples/03_juno.md`：技术型科普文

建议后续新增案例时，也保留：

- 原始热点事实
- 内容类型判断
- 小红书发布文案
- 轮播图结构
- 图片提示词
- AI 味自查评分
- 修改前后差异说明
- 合规边界

每条热点建议按以下流程处理：

```text

→ 选题判断：发现型 / 机制型 / 技术型 / 争议反常识型

→ 初稿生成
→ 去 AI 味人工编辑
→ AI 味自查评分
→ 不达标重写
→ 标题与正文定稿
→ 5 张轮播图规划
```

## 四类内容方向

### 1. 发现型科普文

适合：

- 古生物发现
- 新物种发现
- 天文发现
- 海洋、地质、生态、环境发现
- 考古、化石、自然奇观

重点：

- 讲清楚发现了什么
- 为什么这个发现重要
- 它修正了过去哪种认识
- 它补上了哪个知识空白
- 不强行引申到教育或课程

---

### 2. 机制型科普文

适合：

- 大脑、记忆、睡眠
- 免疫、代谢、衰老
- 肌肉、营养、运动
- 疾病机制
- 生命过程和人体机制

重点：

- 用大白话解释机制
- 保留研究对象、实验条件和边界
- 不把机制研究写成立刻可用的生活建议
- 不制造焦虑，不做治疗承诺

---

### 3. 技术型科普文

适合：

- AI + 科研
- 新材料
- 超导、芯片、机器人
- 大科学装置
- 生物技术、基因技术、医学技术

重点：

- 讲清技术解决了什么问题
- 原来难在哪里
- 新方法新在哪里
- 现在还处在哪个阶段
- 不夸大成“马上改变生活”

---

### 4. 争议 / 反常识科普文

适合：

- 推翻旧认知
- 修正经典理论
- 与大众常识相反的研究
- 容易被误读、标题党化的科学新闻

重点：

- 先讲大众原本怎么理解
- 再讲新研究到底观察到了什么
- 明确哪些是事实，哪些只是推测
- 防止过度解读

## 视觉风格

图片统一遵循：

- 小红书 3:4 竖版
- 5 张独立图，不拼成长图
- 专业、可信、现代、清爽、轻科技感
- 深蓝 / 科技蓝 / 白底 / 少量亮黄
- 信息图风格，每页一个核心结论
- 不做低级海报感
- 不做电商风
- 不做纯装饰图
- 不做过度炫光、虚假科技感

详细规范见：

```text
docs/visual_style_guide.md
assets/ASSET_INDEX.md
```

## 合规提醒

健康类内容必须避免：

- 医疗诊断
- 治疗承诺
- “运动防癌”“逆转疾病”等绝对化说法
- 把动物模型、体外实验、机制研究直接外推到人体结论
- 用科研结果制造健康焦虑

科学类内容必须避免：

- 为了传播效果牺牲准确性
- 把“研究提示”写成“已经证明”
- 使用无法核准的夸张表述
- 把“修正认知”写成“彻底推翻”

图片类内容必须避免：

- 未授权直接搬运论文图、机构图或新闻图
- 恐吓式医疗画面
- 低幼卡通化处理严肃科学议题
- 文字过密导致手机端不可读

更多规则见：

```text
docs/source_and_compliance.md
```
