# Asset Index｜视觉参考图索引

本文件记录当前仓库 `assets/` 目录中的视觉参考图，用于后续保持小红书科普轮播图风格一致。

本版本已从“教育 / 品牌转化型参考图索引”调整为“纯科普转译型参考图索引”。

---

## 使用说明

- 这些图片是风格参考资产，不建议每次直接复用原图。
- 后续生成新图时，应参考其版式、颜色、信息层级、标题层级、画面密度和表达方式。
- 新增图片后，请同步更新本文件。
- 如果 `assets/` 中存在近期发布图，优先参考近期发布图；旧资产仅作为补充风格参考。
- 不要因为旧资产中出现“教育 / 健康管理 / AI时代人才”等字样，就强行把新选题引向教育、课程或品牌转化。

---

## 视觉参考优先级

生成新图时，优先级如下：

```text
1. 近期发布图组
2. 与当前内容类型最接近的旧资产
3. 同类科学题材的结构参考
4. 通用深蓝科技 / 信息图风格参考
```

如果近期发布图与当前选题风格冲突，以内容准确表达为准，不要为了统一风格牺牲科学信息。

# 七、图片生成前必须完成的视觉参考选择

每次正式生成图片前，必须先输出：

```text
本次视觉参考选择：

内容类型：
优先参考图：
参考原因：
需要保持的视觉特征：
需要避免的偏差：
```

示例：

```text
本次视觉参考选择：

内容类型：机制型科普文

优先参考图：
- ChatGPT Image 2026年7月6日 16_32_09 (1).png
- 健康管理的多维度解读.png
- 肌肉信号传递与衰老对比.png

参考原因：
当前选题属于大脑记忆机制研究，需要参考近期发布图组的最新账号样式，同时结合机制型科普图的信息结构。

需要保持的视觉特征：
深蓝科技背景、白色大标题、亮黄色强调关键数字、机制拆解、时间轴、信息卡片、手机端可读。

需要避免的偏差：
不要真实患者照片，不要手术室画面，不要恐怖电极植入图，不要医疗广告感，不要低幼卡通，不要直接搬运论文图。
```

# 十、快速选择规则

如果不知道选哪张参考图，按下面规则：

- 自然发现、古鱼、蓝洞、鲸落、化石 → 先选“发现型科普参考图”
- 大脑、睡眠、记忆、肌肉、衰老、健康机制 → 先选“机制型科普参考图”
- AI、超导、材料、JUNO、大科学装置 → 先选“技术型科普参考图”
- 推翻旧认知、修正理论、容易被误读 → 先选“争议 / 反常识科普参考图”
- 需要统一最新账号风格 → 先选“近期发布图组”

默认策略：

```text
近期发布图组 + 当前内容类型最接近的旧资产 = 最稳妥的参考组合
```
